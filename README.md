# mlflow-server

mlflow项目地址
https://github.com/mlflow/mlflow

# 启动

https://hub.docker.com/r/napoler/mlflow-server

```bash
docker push napoler/mlflow-server
```

# 打包

```bash
docker build -t mlflow-server .
```
